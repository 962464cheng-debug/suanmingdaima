# 超管助手 - 移动应用设计文档

## 应用概述

超管助手是一款用于教育和安全研究目的的Android超级管理工具，针对麻将游戏应用(com.leyou.ruianmahjong v2.3.5)的20多个安全漏洞进行测试和验证。应用采用竖屏设计(9:16)，支持单手操作，遵循iOS Human Interface Guidelines设计规范。

## 核心设计原则

本应用遵循主流iOS移动应用设计标准，界面简洁专业，功能模块化清晰。配色方案采用深蓝色(#1E3A8A)为主色调，电光青色(#06B6D4)为强调色，营造专业可信的技术氛围。所有界面元素针对移动端触控操作优化，确保单手可达性和操作流畅性。

## 屏幕列表与功能

### 1. 首页仪表板 (Home Dashboard)
**主要内容**: 显示目标应用信息、漏洞总览统计、快速操作入口
**功能模块**:
- 目标应用信息卡片(包名、版本号、安装状态)
- 漏洞统计卡片(总数、分类统计、严重程度分布)
- 8个功能模块快速入口(网格布局)
- 实时状态指示器(连接状态、监控状态)

### 2. 网络监控 (Network Monitor)
**主要内容**: HTTP/HTTPS请求列表、请求详情、实时拦截开关
**功能模块**:
- 请求列表(时间、URL、方法、状态码)
- 请求详情查看(请求头、请求体、响应体)
- 拦截规则配置
- 请求修改和重放功能
- 导出抓包数据

### 3. 数据透视 (Data Inspector)
**主要内容**: SharedPreferences数据、数据库内容、文件系统浏览
**功能模块**:
- SharedPreferences键值对列表
- 数据编辑和保存
- 数据库表查看(SQLite)
- 文件浏览器
- 数据导出功能

### 4. 权限管理 (Permission Manager)
**主要内容**: 应用权限列表、权限状态、权限监控日志
**功能模块**:
- 权限列表(位置、录音、摄像头、通讯录、短信等)
- 权限状态切换
- 权限调用日志
- 权限危险等级标识

### 5. 支付破解 (Payment Cracker)
**主要内容**: 支付流程拦截、支付结果伪造、订单管理
**功能模块**:
- 支付拦截开关
- 支付金额修改器
- 支付结果伪造(成功/失败)
- 订单ID生成器
- 支付历史记录

### 6. 组件管理 (Component Manager)
**主要内容**: 导出组件列表(Activity/Service/Receiver/Provider)
**功能模块**:
- 组件列表(分类显示)
- 组件详情(Intent Filter、权限要求)
- 组件启动工具
- 导出组件标识

### 7. 凭证提取 (Credential Extractor)
**主要内容**: 硬编码凭证列表、API密钥、证书信息
**功能模块**:
- 凭证列表(高德地图、微信、Bugly、闲聊)
- 凭证详情查看
- 一键复制功能
- 凭证导出

### 8. 脚本注入 (Script Injector)
**主要内容**: Lua脚本编辑器、脚本库、执行日志
**功能模块**:
- 脚本编辑器(代码高亮)
- 脚本库管理
- 脚本执行
- 执行结果查看

### 9. 日志分析 (Log Analyzer)
**主要内容**: 实时日志流、日志过滤、崩溃日志
**功能模块**:
- 实时日志显示
- 日志级别过滤(Verbose/Debug/Info/Warn/Error)
- 关键词搜索
- 日志导出

### 10. 设置页面 (Settings)
**主要内容**: 应用配置、主题切换、关于信息
**功能模块**:
- 目标应用配置
- 深色模式切换
- 日志保留设置
- 法律声明
- 版本信息

## 关键用户流程

### 流程1: 网络拦截与修改
1. 用户打开首页 → 点击"网络监控"卡片
2. 进入网络监控页面 → 开启拦截开关
3. 列表显示实时请求 → 点击某个请求
4. 查看请求详情 → 点击"修改"按钮
5. 修改参数 → 点击"重放"按钮
6. 查看修改后的响应结果

### 流程2: 支付结果伪造
1. 用户打开首页 → 点击"支付破解"卡片
2. 进入支付破解页面 → 开启拦截开关
3. 在目标应用中发起支付
4. 返回超管助手 → 显示拦截到的支付请求
5. 选择"伪造成功" → 点击"注入"按钮
6. 目标应用收到伪造的支付成功回调

### 流程3: 数据查看与修改
1. 用户打开首页 → 点击"数据透视"卡片
2. 进入数据透视页面 → 选择"SharedPreferences"标签
3. 查看ChInfo.xml数据 → 点击某个键值对
4. 修改值 → 点击"保存"按钮
5. 显示保存成功提示

### 流程4: 凭证提取
1. 用户打开首页 → 点击"凭证提取"卡片
2. 进入凭证提取页面 → 显示所有硬编码凭证
3. 点击某个凭证(如高德地图API密钥)
4. 查看凭证详情 → 点击"复制"按钮
5. 显示复制成功提示

## 配色方案

### 主色调
- **主色(Primary)**: #1E3A8A (深蓝色) - 用于主要按钮、强调元素
- **强调色(Accent)**: #06B6D4 (电光青色) - 用于活动状态、高亮元素

### 功能色
- **成功(Success)**: #22C55E - 用于成功状态、正常运行指示
- **警告(Warning)**: #F59E0B - 用于警告提示、中等风险
- **错误(Error)**: #EF4444 - 用于错误提示、高风险漏洞
- **信息(Info)**: #3B82F6 - 用于信息提示

### 中性色
- **背景(Background)**: #FFFFFF (浅色) / #151718 (深色)
- **表面(Surface)**: #F5F5F5 (浅色) / #1E2022 (深色)
- **前景(Foreground)**: #11181C (浅色) / #ECEDEE (深色)
- **次要文本(Muted)**: #687076 (浅色) / #9BA1A6 (深色)
- **边框(Border)**: #E5E7EB (浅色) / #334155 (深色)

## 视觉设计特点

### 卡片设计
所有功能模块采用圆角卡片设计(border-radius: 16px)，卡片内包含图标、标题、描述和状态指示器。卡片支持点击反馈(opacity: 0.7)，提供良好的交互体验。

### 图标系统
使用Material Icons图标库，确保跨平台一致性。每个功能模块配备专属图标，图标尺寸统一为24-28px，颜色根据主题自动适配。

### 排版系统
- **标题1**: 32px, Bold - 用于页面主标题
- **标题2**: 24px, Semibold - 用于卡片标题
- **标题3**: 18px, Semibold - 用于列表项标题
- **正文**: 16px, Regular - 用于正文内容
- **说明**: 14px, Regular - 用于辅助说明
- **标签**: 12px, Medium - 用于标签和状态

### 间距系统
采用8px基准网格系统:
- **xs**: 4px - 用于紧密元素
- **sm**: 8px - 用于相关元素
- **md**: 16px - 用于卡片内边距
- **lg**: 24px - 用于卡片间距
- **xl**: 32px - 用于页面边距

## 交互设计

### 触控反馈
所有可点击元素提供视觉反馈(opacity或scale变化)和触觉反馈(haptic feedback)。主要操作按钮使用scale: 0.97的轻微缩放效果，次要操作使用opacity: 0.7的透明度变化。

### 加载状态
数据加载时显示骨架屏或加载指示器，避免空白页面。长时间操作显示进度条，提供操作取消选项。

### 错误处理
所有错误信息以Toast或Alert形式展示，提供明确的错误原因和解决建议。关键操作前显示确认对话框，防止误操作。

## 法律和道德声明

应用启动时显示法律声明弹窗，用户必须同意后才能使用。声明内容包括:

- 使用者需自行承担所有法律责任

## 技术实现说明

由于无Root权限限制，部分功能采用以下技术方案:
- **网络拦截**: 使用本地VPN或HTTP代理技术
- **数据访问**: 通过ADB接口或应用级API
- **权限管理**: 使用反射机制访问系统API
- **支付破解**: 通过Intent劫持和Activity启动
- **脚本注入**: 使用Frida动态注入框架

所有功能均在合法权限范围内实现，不涉及系统级漏洞利用。
